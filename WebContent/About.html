Why CollegeFests.com :

It has been very difficult to the event coordinators to file the registrations their event and set the nodel points to collect the info regarding the participants.

So collegeFests.com has made things easy to both the participants and also the event coordinators.Here all the event details can be published which can be viewed by all and intrested people can register to the event by providing their details.Event coordinators can directly provide the url of the event to the people for easy registration

How Does Things Workout Here :

There are two links you can see "PUBLISH AN EVENT" and "EVENTS" with their own purposes.

The publish Link can be used only by the registered publishers and there can be atmost one registered publisher to a college.Once the registration is done then all the events regarding to that college will be published by him

The Events link does not require any registrations.The user can directly check out the most recent events snd can easily get registered by clicking "Register Here" button.All the details of the registered participants will be sent to the event coordinator two days before the event
